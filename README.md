# Электронная библиотека

## Описание проекта

Веб-приложение «Электронная библиотека» разработано в рамках экзаменационного задания.

Проект предназначен для ведения реестра книг, просмотра информации о книгах, загрузки обложек, создания рецензий и модерации пользовательских отзывов.

# Функциональность системы

## Просмотр книг

На главной странице отображается список книг. Для каждой книги выводятся название, жанры, год издания, средняя оценка и количество рецензий.

## Добавление книг

Пользователь с ролью администратора может добавлять новые книги в систему. При добавлении указываются основные данные книги и загружается обложка.

## Редактирование книг

Пользователи с ролями администратора и модератора могут редактировать данные книг.

## Удаление книг

Пользователь с ролью администратора может удалять книги. При удалении книги также удаляются связанные с ней данные: обложка, рецензии и связи с жанрами.

## Просмотр страницы книги

На странице книги отображаются обложка, название, описание, год издания, издательство, автор, объём в страницах и жанры. 

## Рецензии

Авторизованные пользователи могут оставлять рецензии на книги. В рецензии указывается оценка и текст отзыва.

## Модерация рецензий

Вариант №1 предусматривает функциональность модерации рецензий.

После создания новая рецензия получает статус «На рассмотрении». На странице книги отображаются только рецензии со статусом «Одобрена».

Модератор может просматривать список рецензий, находящихся на рассмотрении, и принимать решение: одобрить или отклонить рецензию.

## Мои рецензии

Авторизованный пользователь может открыть страницу «Мои рецензии» и посмотреть список всех своих рецензий с указанием книги, оценки, текста и текущего статуса.

# Роли пользователей

В системе используются три роли:

## Администратор

Администратор имеет полный доступ к системе. Он может добавлять, редактировать и удалять книги.

## Модератор

Модератор может редактировать книги и выполнять модерацию рецензий.

## Пользователь

Пользователь может просматривать книги и оставлять рецензии.
